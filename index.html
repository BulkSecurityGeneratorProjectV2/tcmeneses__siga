Visão Geral

O Projeto SIGA é a versão de código aberto (GPL) do Sistema Integrado de Gestão Administrativa - SIGA, desenvolvido pela Seção Judiciária do Rio de Janeiro - SJRJ.

Clique aqui para fazer o download dos slides da apresentação do SIGA na IV Reunião dos Arquivos Judicias do Brasil.

Características Gerais

  • Módulos de gestão documental, gestão de identidade, workflow, serviços e base de conhecimento
  • Controla documentos físicos e digitais
  • Suporte a assinatura digital
  • Desenvolvido em Java

Instalação do SIGA

Para acompanhar as versões do SIGA veja esta documentação. Para instalar o SIGA leia esta documentação Utilizar o jdk 6 para compilar o projeto. Alguns erros foram encontrados ao tentar compilar o projeto com o jdk 7. Usuários Windows: Executar lib/configure.bat para instalar dependências adicionais no repositório maven local. Usuários Linux/Mac: Executar lib/configure.sh para instalar dependências adicionais no repositório maven local. Compilação: Utilizar o comando "mvn clean instal". Os war necessários para rodar a aplicação vão ser criados do diretório target/.
